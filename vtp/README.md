# VTP Configuration Lab

## Objective
Configure VTP to propagate VLAN information across switches.

## Tasks
1. Create VLANs on SW1.
2. Configure trunk links between SW1–SW2–SW3.
3. Configure VTP:
   - Domain: ccn.com
   - Password: ccn@123
   - SW1: Server mode
   - SW2 & SW3: Client mode
4. Verify VLAN propagation.

## Files
- `configuration.txt`
- `verification.txt`
